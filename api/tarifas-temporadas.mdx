Manage seasonal pricing strategies with dynamic rates based on time periods and room categories.

## Overview

The SGRH system implements flexible pricing through:

- **Temporadas (Seasons)**: Time periods with specific pricing rules
- **Tarifas (Rates)**: Category-specific prices for each season

This allows the hotel to implement high/low season pricing, special events, holidays, etc.

## Temporada Object

A temporada (season) defines a time period with associated pricing.

### Attributes

<ResponseField name="temporadaId" type="number">
  Unique identifier for the season
</ResponseField>

<ResponseField name="nombreTemporada" type="string">
  Season name (max 50 characters)
</ResponseField>

<ResponseField name="fechaInicio" type="string">
  Start date of the season (ISO 8601)
</ResponseField>

<ResponseField name="fechaFin" type="string">
  End date of the season (ISO 8601)
</ResponseField>

## Tarifa Object

A tarifa (rate) defines pricing for a specific room category during a season.

### Attributes

<ResponseField name="tarifaTemporadaId" type="number">
  Unique identifier for the rate
</ResponseField>

<ResponseField name="categoriaHabitacionId" type="number">
  Room category ID
</ResponseField>

<ResponseField name="temporadaId" type="number">
  Season ID
</ResponseField>

<ResponseField name="precio" type="number">
  Price for this category during this season
</ResponseField>

## List Temporadas

Retrieve all seasons with their associated rates.

### Endpoint

```http
GET /api/temporadas
```

### Query Parameters

<ParamField query="pageNumber" type="number" default="1">
  Page number for pagination
</ParamField>

<ParamField query="pageSize" type="number" default="20">
  Number of items per page
</ParamField>

<ParamField query="activas" type="boolean">

</ParamField>

<ParamField query="fecha" type="string">

## Create Temporada

Create a new season.

### Endpoint

```http
POST /api/temporadas
```

### Request Body

<ParamField body="nombreTemporada" type="string" required>
  Season name (max 50 characters)
</ParamField>

<ParamField body="fechaInicio" type="string" required>
  Start date (ISO 8601 format: YYYY-MM-DD)
</ParamField>

<ParamField body="fechaFin" type="string" required>
  End date (must be after start date)
</ParamField>

<Warning>
Seasons cannot overlap. Ensure the date range doesn't conflict with existing seasons.
</Warning>

### Request Example

<CodeGroup>
```bash cURL
curl -X POST https://api.sgrh.com/api/temporadas \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "nombreTemporada": "Semana Santa 2024",
    "fechaInicio": "2024-03-24",
    "fechaFin": "2024-03-31"
  }'
```

```csharp C#
var nuevaTemporada = new
{
    NombreTemporada = "Semana Santa 2024",
    FechaInicio = "2024-03-24",
    FechaFin = "2024-03-31"
};

var content = new StringContent(
    JsonSerializer.Serialize(nuevaTemporada),
    Encoding.UTF8,
    "application/json"
);

var response = await client.PostAsync(
    "https://api.sgrh.com/api/temporadas",
    content
);
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "temporadaId": 3,
    "nombreTemporada": "Semana Santa 2024",
    "fechaInicio": "2024-03-24",
    "fechaFin": "2024-03-31"
  },
  "message": "Temporada creada exitosamente"
}
```

## Get Applicable Rate

Get the applicable rate for a room category on a specific date.

### Endpoint

```http
GET /api/tarifas/aplicable
```

### Query Parameters

<ParamField query="categoriaHabitacionId" type="number" required>
  Room category ID
</ParamField>

<ParamField query="fecha" type="string" required>
  Date to check (ISO 8601 format)
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sgrh.com/api/tarifas/aplicable?categoriaHabitacionId=1&fecha=2024-12-25" \
  -H "Authorization: Bearer {token}"
```

```csharp C#
var query = "?categoriaHabitacionId=1&fecha=2024-12-25";
var response = await client.GetAsync(
    $"https://api.sgrh.com/api/tarifas/aplicable{query}"
);
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "categoriaHabitacionId": 1,
    "nombreCategoria": "Suite",
    "fecha": "2024-12-25",
    "temporadaId": 1,
    "nombreTemporada": "Temporada Alta",
    "precioBase": 250.00,
    "precioAplicado": 350.00,
    "tarifaTemporadaId": 1
  }
}
```

<Note>
If no season covers the specified date, the `precioBase` (base price) from the room category is returned with `temporadaId` and `tarifaTemporadaId` set to `null`.
</Note>

## Error Responses

### Overlapping Seasons

```json
{
  "success": false,
  "errors": [
    {
      "code": "TEMPORADA_OVERLAP",
      "message": "El rango de fechas se solapa con la temporada existente: Temporada Alta"
    }
  ],
  "message": "Business rule violation"
}
```

### Duplicate Rate

```json
{
  "success": false,
  "errors": [
    {
      "code": "DUPLICATE_TARIFA",
      "message": "Ya existe una tarifa para esta categoría en esta temporada"
    }
  ],
  "message": "Conflict"
}
```

### Invalid Date Range

```json
{
  "success": false,
  "errors": [
    {
      "code": "INVALID_DATE_RANGE",
      "message": "La fecha de fin debe ser posterior a la fecha de inicio",
      "field": "fechaFin"
    }
  ],
  "message": "Validation failed"
}
```
