Generate comprehensive reports for business analysis including reservations, occupancy, and revenue.

## Overview

The SGRH reporting module provides insights into hotel operations through three main report types:

- **Reservations Report**: Analysis of reservation patterns and statistics
- **Occupancy Report**: Room occupancy rates and availability trends
- **Revenue Report**: Financial performance and income analysis

All reports support flexible date ranges and various grouping options.

## Common Report Parameters

All report endpoints share common query parameters:

<ParamField query="fechaInicio" type="string" required>
  Start date for the report period (ISO 8601 format: YYYY-MM-DD)
</ParamField>

<ParamField query="fechaFin" type="string" required>
  End date for the report period (ISO 8601 format: YYYY-MM-DD)
</ParamField>

<ParamField query="agrupacion" type="string" default="Dia">
  Grouping/aggregation level: `Dia`, `Semana`, `Mes`, `Año`
</ParamField>

## Reservations Report

Generate a detailed report of reservations for a specified period.

### Endpoint

```http
GET /api/reportes/reservas
```

### Query Parameters

<ParamField query="fechaInicio" type="string" required>
  Report start date
</ParamField>

<ParamField query="fechaFin" type="string" required>
  Report end date
</ParamField>

<ParamField query="agrupacion" type="string" default="Dia">
  Grouping: `Dia`, `Semana`, `Mes`, `Año`
</ParamField>

<ParamField query="estado" type="string">
  Filter by reservation state: `Pendiente`, `Confirmada`, `Cancelada`, `Finalizada`
</ParamField>

<ParamField query="categoriaHabitacionId" type="number">
  Filter by room category
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sgrh.com/api/reportes/reservas?fechaInicio=2024-01-01&fechaFin=2024-03-31&agrupacion=Mes" \
  -H "Authorization: Bearer {token}"
```

```csharp C#
var query = "?fechaInicio=2024-01-01&fechaFin=2024-03-31&agrupacion=Mes";
var response = await client.GetAsync(
    $"https://api.sgrh.com/api/reportes/reservas{query}"
);
```
</CodeGroup>

## Export Report

Export a report in various formats.

### Endpoint

```http
GET /api/reportes/{tipo}/export
```

### Path Parameters

<ParamField path="tipo" type="string" required>
  Report type: `reservas`, `ocupacion`, `ingresos`
</ParamField>

### Query Parameters

Same as the respective report endpoint, plus:

<ParamField query="formato" type="string" default="pdf">
  Export format: `pdf`, `excel`, `csv`
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sgrh.com/api/reportes/ingresos/export?fechaInicio=2024-01-01&fechaFin=2024-12-31&formato=excel" \
  -H "Authorization: Bearer {token}" \
  --output reporte-ingresos-2024.xlsx
```

```csharp C#
var query = "?fechaInicio=2024-01-01&fechaFin=2024-12-31&formato=excel";
var response = await client.GetAsync(
    $"https://api.sgrh.com/api/reportes/ingresos/export{query}"
);

if (response.IsSuccessStatusCode)
{
    var fileBytes = await response.Content.ReadAsByteArrayAsync();
    await File.WriteAllBytesAsync("reporte-ingresos-2024.xlsx", fileBytes);
}
```
</CodeGroup>

### Response

Returns the file as binary data with appropriate `Content-Type` header:

- **PDF**: `application/pdf`
- **Excel**: `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`
- **CSV**: `text/csv`

## Report Metrics Glossary

### Occupancy Metrics

<ResponseField name="tasaOcupacion">
  **Occupancy Rate**: Percentage of occupied rooms
  
  Formula: `(Habitaciones Ocupadas / Habitaciones Disponibles) × 100`
</ResponseField>

<ResponseField name="revPAR">
  **Revenue Per Available Room**: Average revenue per available room
  
  Formula: `Ingreso Total Habitaciones / Total Habitaciones Disponibles`
</ResponseField>

<ResponseField name="ADR">
  **Average Daily Rate**: Average price per occupied room
  
  Formula: `Ingreso Habitaciones / Habitaciones Ocupadas`
</ResponseField>

### Reservation Metrics

<ResponseField name="tasaCancelacion">
  **Cancellation Rate**: Percentage of cancelled reservations
  
  Formula: `(Reservas Canceladas / Total Reservas) × 100`
</ResponseField>

<ResponseField name="promedioEstadia">
  **Average Length of Stay**: Average nights per reservation
  
  Formula: `Total Noches / Total Reservas`
</ResponseField>

## Authorization

<Warning>
**Role Requirements**

- **Reservations Report**: Available to `Administrador` and `Recepcionista`
- **Occupancy Report**: Available to `Administrador` and `Recepcionista`
- **Revenue Report**: Available to `Administrador` only
- **Export Reports**: Available to `Administrador` only
</Warning>
