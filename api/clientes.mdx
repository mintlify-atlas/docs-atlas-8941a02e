Manage hotel clients including registration, updates, search, and deletion.

## Client Object

The client object represents a hotel guest with their personal information.

### Attributes

<ResponseField name="clienteId" type="number">
  Unique identifier for the client
</ResponseField>

<ResponseField name="nationalId" type="string">
  National identification number (max 20 characters)
</ResponseField>

<ResponseField name="nombreCliente" type="string">
  Client's first name (max 100 characters)
</ResponseField>

<ResponseField name="apellidoCliente" type="string">
  Client's last name (max 100 characters)
</ResponseField>

<ResponseField name="email" type="string">
  Client's email address (max 100 characters)
</ResponseField>

<ResponseField name="telefono" type="string">
  Client's phone number (max 20 characters)
</ResponseField>

<ResponseField name="fechaRegistro" type="string">
  Registration date in ISO 8601 format (UTC)
</ResponseField>

## List Clients

Retrieve a paginated list of clients with optional filtering.

### Endpoint

```http
GET /api/clientes
```

### Query Parameters

<ParamField query="pageNumber" type="number" default="1">
  Page number for pagination
</ParamField>

<ParamField query="pageSize" type="number" default="20">
  Number of items per page (max 100)
</ParamField>

<ParamField query="searchTerm" type="string">
  Search by name, email, or national ID
</ParamField>

<ParamField query="nationalId" type="string">
  Filter by specific national ID
</ParamField>

<ParamField query="email" type="string">
  Filter by email address
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.sgrh.com/api/clientes?pageNumber=1&pageSize=20&searchTerm=Garcia" \
  -H "Authorization: Bearer {token}"
```

```csharp C#
using System.Net.Http;

var client = new HttpClient();
client.DefaultRequestHeaders.Authorization = 
    new AuthenticationHeaderValue("Bearer", token);

var query = "?pageNumber=1&pageSize=20&searchTerm=Garcia";
var response = await client.GetAsync(
    $"https://api.sgrh.com/api/clientes{query}"
);

var result = await response.Content.ReadAsStringAsync();
```
</CodeGroup>

### Response

## Create Client

Register a new client in the system.

### Endpoint

```http
POST /api/clientes
```

### Request Body

<ParamField body="nationalId" type="string" required>
  National identification number (max 20 characters, must be unique)
</ParamField>

<ParamField body="nombreCliente" type="string" required>
  Client's first name (max 100 characters)
</ParamField>

<ParamField body="apellidoCliente" type="string" required>
  Client's last name (max 100 characters)
</ParamField>

<ParamField body="email" type="string" required>
  Valid email address (max 100 characters, must be unique)
</ParamField>

<ParamField body="telefono" type="string" required>
  Phone number (max 20 characters)
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X POST https://api.sgrh.com/api/clientes \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "nationalId": "001-9876543-2",
    "nombreCliente": "Maria",
    "apellidoCliente": "Rodriguez",
    "email": "maria.rodriguez@email.com",
    "telefono": "+1-809-555-9876"
  }'
```

```csharp C#
var nuevoCliente = new
{
    NationalId = "001-9876543-2",
    NombreCliente = "Maria",
    ApellidoCliente = "Rodriguez",
    Email = "maria.rodriguez@email.com",
    Telefono = "+1-809-555-9876"
};

var content = new StringContent(
    JsonSerializer.Serialize(nuevoCliente),
    Encoding.UTF8,
    "application/json"
);

var response = await client.PostAsync(
    "https://api.sgrh.com/api/clientes",
    content
);
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "clienteId": 2,
    "nationalId": "001-9876543-2",
    "nombreCliente": "Maria",
    "apellidoCliente": "Rodriguez",
    "email": "maria.rodriguez@email.com",
    "telefono": "+1-809-555-9876",
    "fechaRegistro": "2024-03-15T14:30:00Z"
  },
  "message": "Cliente creado exitosamente"
}
```

## Update Client

Update an existing client's information.

### Endpoint

```http
PUT /api/clientes/{id}
```

### Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the client to update
</ParamField>

### Request Body

<ParamField body="nombreCliente" type="string" required>
  Client's first name (max 100 characters)
</ParamField>

<ParamField body="apellidoCliente" type="string" required>
  Client's last name (max 100 characters)
</ParamField>

<ParamField body="email" type="string" required>
  Valid email address (max 100 characters)
</ParamField>

<ParamField body="telefono" type="string" required>
  Phone number (max 20 characters)
</ParamField>

<Note>
The `nationalId` cannot be updated once a client is created.
</Note>

### Request Example

<CodeGroup>
```bash cURL
curl -X PUT https://api.sgrh.com/api/clientes/2 \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "nombreCliente": "Maria Isabel",
    "apellidoCliente": "Rodriguez",
    "email": "maria.isabel@email.com",
    "telefono": "+1-809-555-9999"
  }'
```

```csharp C#
var actualizarCliente = new
{
    NombreCliente = "Maria Isabel",
    ApellidoCliente = "Rodriguez",
    Email = "maria.isabel@email.com",
    Telefono = "+1-809-555-9999"
};

var content = new StringContent(
    JsonSerializer.Serialize(actualizarCliente),
    Encoding.UTF8,
    "application/json"
);

var response = await client.PutAsync(
    $"https://api.sgrh.com/api/clientes/{clienteId}",
    content
);
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "clienteId": 2,
    "nationalId": "001-9876543-2",
    "nombreCliente": "Maria Isabel",
    "apellidoCliente": "Rodriguez",
    "email": "maria.isabel@email.com",
    "telefono": "+1-809-555-9999",
    "fechaRegistro": "2024-03-15T14:30:00Z"
  },
  "message": "Cliente actualizado exitosamente"
}
```

## Delete Client

Delete a client from the system.

### Endpoint

```http
DELETE /api/clientes/{id}
```

### Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the client to delete
</ParamField>

<Warning>
A client cannot be deleted if they have active reservations. Cancel or complete all reservations first.
</Warning>

### Request Example

<CodeGroup>
```bash cURL
curl -X DELETE https://api.sgrh.com/api/clientes/2 \
  -H "Authorization: Bearer {token}"
```

```csharp C#
var response = await client.DeleteAsync(
    $"https://api.sgrh.com/api/clientes/{clienteId}"
);
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "message": "Cliente eliminado exitosamente"
}
```

```json
  "success": false,
  "errors": [
    {
      "code": "VALIDATION_ERROR",
      "message": "El email es requerido",
      "field": "email"
    }
  ],
  "message": "Validation failed"
}
```

### Duplicate Client

```json
{
  "success": false,
  "errors": [
    {
      "code": "DUPLICATE_CLIENT",
      "message": "Ya existe un cliente con este nationalId o email"
    }
  ],
  "message": "Conflict"
}
```

### Client Not Found

```json
{
  "success": false,
  "errors": [
    {
      "code": "NOT_FOUND",
      "message": "Cliente no encontrado"
    }
  ],
  "message": "Resource not found"
}
```

### Cannot Delete (Has Reservations)

```json
{
  "success": false,
  "errors": [
    {
      "code": "HAS_ACTIVE_RESERVATIONS",
      "message": "No se puede eliminar un cliente con reservas activas"
    }
  ],
  "message": "Business rule violation"
}
```
